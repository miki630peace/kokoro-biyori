# こころ日和 🐰🎀

気分の波・生理周期・日常生活を1つにまとめて記録できる、じぶんのための手帳アプリ。
マスコットはうさぎの「めろりん」。

## できること

- 🌷 **きもちの記録** — 1日1回、5段階で気分を記録(ラベルは設定画面で自由に変更可)+ひとことメモ
- 🌙 **すいみんの記録** — 就寝時刻・起床時刻から睡眠時間を自動計算
- 💧 **せいりの記録** — 開始/終了をボタンで記録。直近の周期の平均から次回を予測
- 📖 **できごとメモ** — その日にあったことを自由に記録
- 📅 **カレンダー** — 気分・生理・メモ・予測日を月表示で振り返り
- 📈 **グラフ** — 気分の波を折れ線で表示し、生理期間・予測期間・睡眠時間を重ねてトリガーを把握

## 技術メモ

- HTML / CSS / JavaScript のみ(フレームワーク・ビルド不要)
- データはすべてブラウザの localStorage に保存(端末の外には送信されません)
- PWA対応: ホーム画面に追加するとアプリのように使えます(オフライン動作可)
- 設定画面からJSONでバックアップの書き出し/読み込みができます

## 公開方法

静的サイトなので、フォルダごとGitHubにpushしてNetlifyに接続するだけで公開できます。
ビルド設定は不要(publish directory はリポジトリ直下)。

※ 生理の予測はあくまで記録の平均から計算した目安です。
